# Ad-Challenge-FE

Fullstack application built in Nuxt to interact with the backend infrastructure provided by ad-challenge-be.

## Project setup

- Have Ad-Challnge-BE running on your machine, either locally or through docker.

- Install dependencies:

```bash
pnpm install
```

- Run app:

```bash
pnpm dev
```

The app is now reachable at `localhost:3000/`. You can customize the `.env` file to set different backend URIs.
